# 系统性金融风险

## 1.简介
这是一个用于训练并预测系统性金融风险的工程。在该工程中，你可以构造风险指标，
从多种模型中(DWT-Informer,Informer,LSTM,GRU)选择并训练得到自己的预测模型用于系统性金融风险的预测。
